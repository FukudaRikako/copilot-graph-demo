# Microsoft Graph 3D Visualization

Microsoft Graph API の構造をインタラクティブな 3D ビジュアライゼーションで表現するデモアプリケーションです。

## 概要

- **3D グラフビジュアライゼーション**: Three.js を使用した Microsoft Graph API のノード・エッジ構造の可視化
- **ストーリーモード**: スライド形式のプレゼンテーションで Microsoft Graph の各機能を順に紹介
- **インタラクティブ操作**: ノードのホバーやクリックで API パスや詳細情報を表示
- **レイヤー切り替え**: カテゴリごとにノードの表示/非表示を切り替え可能

## 使い方

`index.html` をブラウザで開くだけで動作します。外部サーバーやビルドツールは不要です。

```
# ローカルで開く
start index.html
```

## 技術スタック

- HTML / CSS / JavaScript（単一ファイル構成）
- [Three.js](https://threejs.org/) — 3D レンダリング
